Lesson21 - Event 事件系统

Event 概述

Event 类用于在编辑模式下监听用户的键盘、鼠标等输入事件。与运行时的 Input 类不同，Event 专门用于编辑器拓展开发，在 OnGUI 和 SceneView 回调中使用。

获取当前事件

Event.current

获取当前帧的输入事件。每个 OnGUI 帧都会有一个当前事件。

重要：避免在布局计算期间多次访问 Event.current，应该在函数开头缓存：
